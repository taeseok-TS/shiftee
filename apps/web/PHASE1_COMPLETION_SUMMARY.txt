================================================================================
                        HOW TO ACCESS
================================================================================

FOR ADMIN USERS
1. Log in with ADMIN credentials
2. Navigate to: http://localhost:3000/admin/dashboard
3. Use the red sidebar "A" to navigate
4. Click RoleSwitch dropdown to test as MANAGER or EMPLOYEE

FOR MANAGER USERS
1. Log in with MANAGER credentials
2. Navigate to: http://localhost:3000/manager/dashboard
3. Use the yellow sidebar "M" to navigate
4. Data automatically filtered by your branch

FOR EMPLOYEE USERS
1. Log in with EMPLOYEE credentials
2. Navigate to: http://localhost:3000/dashboard
3. Use the blue sidebar "E" to navigate
4. Data automatically filtered to your personal records
